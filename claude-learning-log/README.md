My journey from zero Node.js → building real backends. This folder is the source of truth and travels with me across laptops.

## How to resume on ANY laptop

> Read `claude-learning-log/` — especially `MENTOR.md`, `progress.md`, and the latest file in `sessions/`. Then continue mentoring me from where I left off, same style.
